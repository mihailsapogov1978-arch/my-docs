# Руководство пользователя по установке и настройке программы {data-toc-label="Установка"}

## Подробная инструкция по настройке параметров конфигурации {data-toc-label="Настройк"}

1. Скачайте последнюю версию.
2. Распакуйте архив.
3. Запустите `app.exe`.

## Конфигурация программы {data-toc-label="Конфигурация"}

Откройте файл `config.json` и укажите:

```json
{
  "language": "ru",
  "theme": "dark"
}
```
